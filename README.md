# QRStudio

핫키
--------
Enter : 설정된 브러시를 선택된 영역에 재할당 합니다.

Esc : 선택된 영역을 해제합니다.

Ctrl + N : 새로운 QR코드를 생성합니다.

Ctrl + S : QR코드를 저장합니다.
